Com o arquivo baixado (recomendo por em Documentos ou Usuários, mas não
na Area de trabalho, devido a possíveis erros de caminho) abra no VS code,
abra o terminal, e digite o comando "npm install" para baixar todos os 
pacotes e dependências (pode vê-los em 'package.json', caso de algum erro e precise instalar manualmente). 

Lembrando que é necessário ter o Node.js e npm instalado.

Nosso projeto usa alem do React Native, o Firebase como banco de dados,
o TailwindCSS com design e o Expo para aplicação.(Precisa ter algum emulador
com Expo Go instalado, para ver a aplicação, eu usei o emulador do android studio msm)

Tanto o Firebase quanto o Tailwaind podem ser necessários baixar manualmente
 com  comando 'nmp'. BAIXAR O TAILWIND NA VERSAO 3.3.2 COM: npm install tailwindcss@3.3.2 (smp de esse comando msm apos os anteriores)


Apos ambiente preparado e o comando "npm install", tudo que é necessário vai estar instalado, restando apenas inicializar o app com "npx expo start",
conseguindo assim mexer no projeto.

**O Firebase é o banco que esta configurado numa conta minha, todo o banco de dados esta montado, mas se alguem quiser mexer em algo se me avisar, que
altero ou passo a conta.
